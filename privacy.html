Privacy Policy for FilterTube

Last updated: 2026

Overview

FilterTube is a Chrome extension designed to filter YouTube content by blocking keywords, hiding Shorts, and removing distractions to improve user focus and productivity.

Data Collection

FilterTube does not collect, store, or transmit any personal data to external servers.

Local Storage Usage

FilterTube uses Chrome's local storage API only to save user preferences such as:

  • Blocked keywords
  • Whitelisted channels
  • Display settings (blur or hide mode)

Data Sharing

No user data is sold, shared, or transferred to third parties.

Remote Code

FilterTube does not use remote code execution. All scripts are included within the extension package.

Third-Party Services

FilterTube does not use any third-party analytics or tracking services.

Contact

If you have any questions about this Privacy Policy, you can contact the developer via the Chrome Web Store listing.
