Privacy Policy

Last updated: June 2026

This Privacy Policy explains how FilterTube operates as a Chrome Extension and how it handles user data.

1. Overview

FilterTube is a browser extension designed to improve YouTube usage by filtering videos based on keywords, blocking unwanted content, and enabling focus features.

The extension is built to work entirely inside your browser and does not require external servers.

2. Data Collection

FilterTube does not collect, store, or transmit any personal data.

The extension processes only:

  • Video titles (on YouTube pages)
  • Channel names
  • User-defined filter keywords
  • User settings stored locally

Important: All processing happens locally inside your browser using content scripts.

3. Permissions Used

FilterTube requests the following Chrome permissions:

  • storage – to save your filter settings locally
  • activeTab – to interact with the current YouTube tab
  • youtube.com access – to read video titles and apply filters

4. Data Storage

All settings (keywords, blocked terms, whitelist channels, and toggles) are stored using chrome.storage.sync.

This data remains inside your Google account's Chrome sync system and is never sent externally.

5. Data Sharing

FilterTube does not share any data with third parties.

6. External Servers

This extension does not use any backend servers. No analytics, tracking scripts, or external APIs are used.

7. Security

Since all processing happens locally, your data never leaves your browser. This minimizes any security risks.

8. Children’s Privacy

FilterTube does not knowingly collect data from anyone, including children under 13.

9. Changes to This Policy

This Privacy Policy may be updated in future versions of the extension. Any changes will be reflected in this document.
